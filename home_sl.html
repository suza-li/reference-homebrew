Welcome to my Homebrew Overview!

This is a site that contains the overview of my homebrew campaign.

What is a homebrew?

A homebrew world is a fantasy world created by the dungeon master or game master not found in any offical books. It usually refers to a game campaign that contains the creator's original content compared to a already published material.

What is this campaign going to be about?

This campaign is going to start in the country of Eldar. The players will be tasked to complete a request from a high figure in the country. How they accomplish the task is for them to discover.
